¡Etiquetas para todos!

Las etiquetas son una característica muy útil para clasificar tu información dentro de EasyBroker. Una etiqueta es una palabra clave o un término que puedes asociar con propiedades, clientes potenciales o contactos.

Una vez que colocas una etiqueta, puedes fácilmente filtrar o buscar algún resultado que coincida con esas etiquetas. Un ejemplo de esto: puedes colocar como etiqueta cuánto es la comisión que recibes si vendes una propiedad, usando etiquetas como “3 por ciento de comisión” o “6 por ciento”. Una vez que colocas las etiquetas en las propiedades, puedes utilizar los filtros de búsqueda para que te aparezcan únicamente las que tienen la etiqueta deseada. Recuerda, las etiquetas funcionan tanto para contactos como para propiedades.

En EasyBroker a menudo utilizamos las etiquetas “posible venta”, “vendedor”, “cliente”, y “candidato para puesto de trabajo” para clasificar a nuestros contactos. Para propiedades, un ejemplo de etiquetas es utilizar la etiqueta “Roma Condesa” o “Distrito Federal” para saber la zona en la que se ubica la propiedad y poder filtrar rápidamente. Ésta es una de las principales ventajas que ofrecen las etiquetas: se adaptan a tu negocio y puedes crear justo las que necesites para que sean una gran herramienta para el manejo de tu inmobiliaria.
