Software CRM inmobiliario para Argentina

Hemos estado revisando qué software de CRM (Customer Relationship Management) para inmobiliarias hay en Argentina ya que tenemos muchos clientes que están en Buenos Aires. No encontramos muchas soluciones en CRM, sin embargo aquí hay una breve muestra de lo hallado. Solo evalué CRMs enfocados en ayudarte a registrar tus comunicaciones dentro y fuera de internet, y que fueron diseñados para funcionar bien en Argentina. Tenían que soportar, por ejemplo, dólares y pesos, y proveer traducciones en Español.

El más grande de los ofrecidos es por mucho Adinco/ArgenProp del grupo Clarín. Adinco es una aplicación para Windows que te permite compartir tu listado de propiedades con otros agentes inmobiliarios, manejar tu inventario, y es el único software que permite publicar tus inmuebles en los sitios web de Clarín como ArgenProp y Buscainmueble. Provee características de CRM, pero muchas inmobiliarias tienden a usarlo principalmente para publicar las propiedades en los sitios de Clarín y usan otras herramientas de software para manejar sus contactos, clientes potenciales y comunicaciones.

Xintel tiene alrededor de 22 años y es una aplicación para Windows. Al parecer, lanzan una nueva versión del software cada dos años y están ahora en la versión 10. Compras el CRM y el software de gestión y luego sólo pagas por actualizaciones cada dos años. Sólo hay un costo mensual si quieres publicar tus inmuebles en tu sitio web y varios portales inmobiliarios gratuitos.

Medrano Soft es un software de gestión completamente offline que corre en Windows. Provee módulos para manejar varios aspectos del negocio, excepto para publicar tus inmuebles online y actualizar tu sitio web. No hay cargos mensuales, sólo compras una licencia de acuerdo con los módulos que necesitas y luego cada tantos años pagas por las actualizaciones. Yo vi un demo en una exposición inmobiliaria y parecía tener varias características para gestionar un negocio inmobiliario pero no pude encontrar mucha información acerca del producto en internet.

Por supuesto, no voy a olvidar nuestro CRM EasyBroker. EasyBroker es un sistema de gestión que está basado completamente en la web e incluye el alojamiento y gestión de un sitio web inmobiliario, también puedes publicar tu listado de inmuebles en muchos portales inmobiliarios gratuitos como harías con Xintel. EasyBroker cobra un pago mensual.

Hubo otras empresas que no me animé a incluir como Inmo-virtual.com, Inmohost, Sistema7G, Maixon y MapaProp que parecen enfocarse en la creación de sitios web y sindicación de propiedades pero ofrecen muy pocas características de un CRM.
